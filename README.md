# Pod2Book Website

Modern, accessible website for Pod2Book - a podcast to eBook converter for the deaf and hard of hearing community.

## Template
Built with [Massively](https://html5up.net/massively) by HTML5 UP

## Features
- Accessibility-focused design
- Responsive layout (works on all devices)
- Clean, modern aesthetic
- Fast loading
- No frameworks or build process needed

## Development
Simply open `index.html` in a browser or use a local server:

```bash
python3 -m http.server 8000
# Visit http://localhost:8000
```

## Deployment
Push to main branch - GitHub Pages will automatically deploy.
